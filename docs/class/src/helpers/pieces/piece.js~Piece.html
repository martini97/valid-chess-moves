    public possiblePositionsInNMoves(movesInfo: Object): AlgebraicNotation[] source

    All the possible (legal) movements of a piece after n moves.

    Params:

    NameTypeAttributeDescription
    movesInfo Object

    Object containing the params.
    movesInfo.startAt Coordinates
    • optional
    • default: this.coordinates

    Starting position of the piece. Defaults to the position of the current piece.
    movesInfo.n Integer
    • optional
    • default: 1

    Number of moves, where 0 < n < 20.
    movesInfo.positions Coordinates[]
    • optional
    • default: []

    Array of positions. Used for recursion.

    Return:

    AlgebraicNotation[]

    Every possible coordinates that the piece can reach, it includes coordinates out of the board.

    Throw:

    BadRequestError

    Throws a BadRequestError if the given n is invalid.

    • Given a Knight should return all valid positions
    • Given a Knight should return all valid positions for it in 2 moves
    • Given a Piece next N moves should always return [] for N > 0
    • Given a Piece next N moves should throw an error for invalid N
